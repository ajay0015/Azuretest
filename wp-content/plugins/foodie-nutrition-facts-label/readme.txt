== Description ==

Foodie Nutrition Facts Label is a plugin that lets you create FDA compliant nutrition facts labels with ease. Whether you're a food blogger or nutritionist, these labels will make your pages pop with some useful scientific information of your recipes. Fill out nutrition values and immediately see how will label look like thanks to real-time preview.

Foodie Nutrition Facts Label does all the calculations for you so all you need to do is fill out values of basic nutrients. Both *Calories from Fat* and *% DV* is calculated automatically.

== Frequently Asked Questions ==

= Installation =

1. Upload the `foodie-nutrition-labels` folder to the `wp-content/plugins` directory.
2. Activate it on `Plugins` screen in WordPress dashboard.

= Do I have to input %DV and Calories from Fat manually? =

Oh no, we wouldn't do that to you! All of *% Daily Value* entries and *Calories from Fat* are calculated automatically as you type the nutrient values in.

= Do labels work with all themes? =

Sure thing! Labels create very tiny footprint and is 100% compatible with your favorite theme.

= Will labels work when I switch themes? =

Absolutely! As long as you keep the plugin active, labels will blend in with just about any theme.
